Afinal, HTML é ou não uma linguagem de programção?

Muito embora haja essa confusão, HTML não configura-se como uma lingaguem de programação. Tendo sido criada na década de 90 pelo britânico Tim Berners-Lee _também criador do WWW_, HTML significa Hipertext Markup Language em tradução livre Lingaguem de Marcação de Hipertexto, que se caracteriza como um componente básico da Web , permitindo assim a inserção de códigos e organizando de informções de uma página Web.
